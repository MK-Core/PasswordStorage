# PasswordStorage
based on 51 Microcontroller

## 硬件型号
  1. 微控制器：AT89S52 (以此为核心构成单片机最小系统)
  2. 液晶显示模块：LCD12864
  3. 4X4矩阵键盘
  4. EEPROM：AT24C64
  5. 串口通讯模块
  6. 两路LED指示灯
  7. 一路蜂鸣器

## 项目功能
  1. 开机核对主密码
    * 主密码输入错误超过最大次数，清空全部数据
  2. 添加新账户
  3. 查看所有账户
    * 修改当前正在查看的账户信息
  4. 修改主密码
    * 核对原密码正确，方可添加新密码
